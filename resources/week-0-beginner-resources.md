# Week 0 Beginner Resources (Exact Links)

Use these in order. Do not try to learn everything in one day.

## Day 1: Computer + File Basics
- Google IT Support basics overview: https://www.coursera.org/professional-certificates/google-it-support
- File path basics (Windows/macOS/Linux concepts): https://edu.gcfglobal.org/en/computerbasics/

## Day 2: Terminal Basics
- Command line crash course (freeCodeCamp): https://www.freecodecamp.org/news/command-line-for-beginners/
- `pwd`, `ls`, `cd` explanations: https://swcarpentry.github.io/shell-novice/

## Day 3: Git + GitHub Basics
- GitHub Skills: Introduction to GitHub: https://skills.github.com/
- Git official book (free): https://git-scm.com/book/en/v2

## Day 4: Push to GitHub
- GitHub Docs: Create a repository: https://docs.github.com/en/repositories/creating-and-managing-repositories/creating-a-new-repository
- GitHub Docs: About remote repositories: https://docs.github.com/en/get-started/git-basics/about-remote-repositories

## Day 5: Python Basics
- Python official tutorial: https://docs.python.org/3/tutorial/
- W3Schools Python basics (quick practice): https://www.w3schools.com/python/

## Day 6: Mini Practice
- Replit (run simple scripts quickly): https://replit.com/
- Practice Python problems: https://www.practicepython.org/

## Day 7: Review
- Repeat your own notes and Week 0 checklist.
- Rebuild your mini project from memory.

## Rule for Using Resources
- 30% learning, 70% hands-on practice.
- After each tutorial, do one small task without looking at the tutorial.
