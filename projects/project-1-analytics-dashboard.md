# Project 1: Analytics Dashboard (Business Intelligence)

## Goal
Build an executive dashboard from raw transactional data and present clear business recommendations.

## Stack
- Python (pandas)
- SQL
- BI tool (Power BI/Tableau/Looker Studio)

## Scope
- Ingest sample sales/customer dataset
- Clean and transform data
- Build KPI layer: revenue, conversion, retention, AOV
- Create dashboard with filters and trend views

## Deliverables
- Data cleaning notebook
- SQL logic file
- Dashboard screenshots or share link
- README with business narrative

## Evaluation Criteria
- Accuracy of metrics
- Clarity of visual design
- Strength of recommendations
